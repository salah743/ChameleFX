# ChameleFX

A compact FX trading stack with:
- FastAPI service (`app/api/server.py`) with autoloaded routers
- Alpha features + portfolio optimizer endpoints
- MT5 resilience (heartbeat/reconnect/outbox with file locks)
- Atomic JSON runtime + simple logging to `chamelefx/runtime/logs/app.log`
